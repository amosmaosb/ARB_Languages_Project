PHP

  • When it was created?

    • PHP was created in 1993 and released in 1995.

  • Who created it (person or group)?

    • PHP was created by Rasmus Lerdorf

  • General history of the language (how has it evolved over time)

    • PHP began development in 1993 when Ramsus wrote programs in C to maintain his personal homepage. He expanded off of that base to create the first version of PHP, called PHP/FI at the time, and release it in 1995. It had much the same basuc functionality that it does today such as the ability to embed HTML. It didn't get the name PHP until the release of PHP 3 in 1997 where it was renamed to PHP: Hypertext Processor which is a recursive acronym.

  • What type of problems the language is typically used for.

    • PHP is mainly used for server-side scripting, this allows for dynamic websites to be created, but is also used fro command-line scripting as well as writing desktop applications with GUI.

  • What is the current state of the language (who is still using it - what for?)

    • PHP is still a pretty popular language due to it being able to create dynamic web applications

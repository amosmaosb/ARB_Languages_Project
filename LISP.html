LISP

  1. When it was created?

    2. LISP was created in 1958 and first appeared in 1960.

  2. Who created it (person or group)?

    3. LISP was created by John McCarthy at MIT.

  3. General history of the language (how has it evolved over time)

    4. John McCarthy began creation of LISP in 1958 and "released" it in 1960 with his paper, it was created as a practical mathematical notation for computer programs influenced by lambda calculus. It was first implemented by Steve Russell on an IBM 704 using ounched cards. During this it was noticed that the "eval" function from McCarthy's paper could be implemented in machine code. This led to a LISP interpreter. A bit later in a1962 the first LISP compiler was made which greatly improved on the speed from the interpreter.

  4. What type of problems the language is typically used for.

    5. Originally LISP was just mathematical notation for computer programs based on lambda calculus but it was later found out that it could be implemented as machine code. The functional nature of the language made it very useful for Artificial Intelligence and that's mainly what it is used for now.

  5. What is the current state of the language (who is still using it - what for?)

    6. Currently, LISP is mostly used to code Artificial Intelligence
